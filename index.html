What can event listeners do?


They allow us to add interactive functionality to HTML elements by “listening” to different events that take place on the page, such as when the user clicks a button, presses a key, or when an element loads.

                            
HTML

<div>
    <h4>Interact with this button</h4>
    <div class="button task_1">Click here</div>
    <p class="action">Waiting for action...</p>
</div>

How to select DOM elements


In order to assign Event Listeners, we need to select the right DOM elements with JS. For that we only need the command: "document.querySelector" for single elements and "document.querySelectorAll" for multiple elements of the same type

You can select elements by:

  • Their id

    document.querySelector("#demo_article");

  • Their classes

    document.querySelector(".tasks_list");

  • Their html tag

    document.querySelector("li");


    We need to use "document.querySelectorAll" to select all the elements of a specific type.

Traversing the DOM


The act of selecting an element from another element.
But why we traverse?
Let’s say you want to go to your neighbor’s house. What’s the fastest and most efficient way to get there?

How do we do it?
let val;
const list = document.querySelector
("ul.task_list");
val = list;

  • Select the children of that element:

    val = list.children;

  • Select a specific children of that element:

    val = list.children[2];

  • Select the children element of that children element:

    val = list.children[2].children;

Traversing the DOM


The act of selecting an element from another element.

How do we do it?
let val;
const listItem = document.querySelector
("li.demo_list");
val = list;

  • Select the parent of that element:

    val = listItem.parentElement;

  • Select a specific attribute of that parent element:

    val = listItem.parentElement.classList;

  • Select that parent element's parent element:

    val = listItem.parentElement.parentElement;

Creating DOM elements from JS


How do we do it?

Create element

Create a list element

const li = document.createElement('li');

Create a link element

const link = document.createElement('a');
Add class
li.className = 'demo_list';
Add id
li.id = "new_item"
Add attribute

Set attributes for the link

link.setAttribute('href', '#');
Create the text node and append
link.appendChild(document.createTextNode('New task'));
Append link to li
li.appendChild(link);
Append li as a child to ul
document.querySelector('ul.tasks_list').appendChild(li);

Replacing DOM elements from JS


How do we do it?

Create an Element
const addedListItem = document.createElement('h5');
Add id
addedListItem.id = 'added-item';
New text node
addedListItem.appendChild(document.createTextNode("I am a heading"));
Select the existing element
const existingListItem = document.querySelector('ul.tasks_list li:nth-child(3)');
Select parent Element
const ulElement = document.querySelector("ul.tasks_list");
Replace
ulElement.replaceChild(addedListItem, existingListItem);

Event properties


Event properties are attributes of a particular event. The values they contain are current for the moment at which the event was triggered.

For instance
  • Select the DOM element
    const article = document.querySelector('article');
  • Add an Event listener
    article.addEventListener('mousemove', runEvent);
  • runEvent function
    function runEvent(e) {
    console.log(`EVENT TYPE: ${e.type}`);
    section.style.backgroundColor = `rgb(${e.offsetX}, ${e.offsetY}, 200)`;
    }
